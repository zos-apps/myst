## About

Explore the mysterious island in this classic adventure game.

## Features

- Puzzle solving
- Exploration
- Rich storyline
- Beautiful graphics

## Installation

### From App Store
Open the **App Store** in zOS and search for "Myst".

### Via npm
```bash
npm install @zos-apps/myst
```

## Usage

- Click to move and interact
- Solve puzzles to progress
- Take notes!



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
